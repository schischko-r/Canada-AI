# Про этот проект

Цель данной работы — создание интерфейса для работы пожарного диспетчера. Интерфейс включает в себя карту пожаров с метками. При клике на метку, отправляется запрос на сервер с нейронной сетью, оценивающей потенциальный ущерб от пожара, после чего, пользователю предлагается сводка относительно местности и приблизительного ущерба от пожара, а так же рекомендации относительно его тушения.
